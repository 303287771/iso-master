ISO Master
=======

A graphical CD/DVD image editor for Linux and BSD systems.

[English]

Overview
----------
ISO Master is an open-source, easy to use graphical CD image editor. You can use this 
program to:
- Add files to an ISO image
- Extract files from an ISO image
- Delete files from an ISO image
- Create bootable ISOs

Supported formats:
- Read: ISO, NRG, MDF
- Write: ISO only

Requirements
------------
- GTK2 development libraries
- Standard build tools (gcc, make)

Compiling
--------
  make

Installing (optional)
-------------------
  sudo make install

Or on Ubuntu/Debian:
  sudo make install

Usage
-----
1. Open an ISO file: File -> Open (Ctrl+O)
2. Add files: Select files from the left pane, click "Add" button
3. Extract files: Select files in the ISO, click "Extract" button
4. Delete files: Select files, press Delete key
5. Save: File -> Save (directly to current file) 
   or File -> Save As (save as new file)

Keyboard shortcuts:
- Ctrl+N: New ISO
- Ctrl+O: Open ISO
- Ctrl+S: Save
- Ctrl+Shift+S: Save As
- F2: Rename
- F3: View
- F4: Edit
- F5: Refresh
- Ctrl+Q: Quit

ISO Master
========

一款适用于 Linux 和 BSD 系统的图形化 CD/DVD 镜像编辑器。

简介
------
ISO Master 是一款开源、易于使用的图形化 CD 镜像编辑器。您可以使用此程序：
- 向 ISO 镜像添加文件
- 从 ISO 镜像提取文件
- 从 ISO 镜像删除文件
- 创建可启动 ISO

支持的格式：
- 读取：ISO、NRG、MDF
- 写入：仅支持 ISO

编译要求
--------
- GTK2 开发库
- 标准编译工具（gcc、make）

编译方法
--------
  make

安装（可选）
-----------
  sudo make install

或在 Ubuntu/Debian 系统上：
  sudo make install

使用方法
--------
1. 打开 ISO 文件：文件 -> 打开（Ctrl+O）
2. 添加文件：从左侧窗格选择文件，点击"添加"按钮
3. 提取文件：在 ISO 中选择文件，点击"提取"按钮
4. 删除文件：选择文件，按 Delete 键
5. 保存：文件 -> 保存（直接保存到当前文件）
   或 文件 -> 另存为（保存为新文件）

快捷键：
- Ctrl+N：新建 ISO
- Ctrl+O：打开 ISO
- Ctrl+S：保存
- Ctrl+Shift+S：另存为
- F2：重命名
- F3：查看
- F4：编辑
- F5：刷新
- Ctrl+Q：退出
